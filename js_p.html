Le role du JAVASCRIPT

JavaScript est un langage de programmation textuel utilisé à la fois côté client et côté serveur qui vous permet de rendre les pages Web interactives. Là où HTML et CSS sont des langages qui donnent une structure et un style aux pages Web, JavaScript donne aux pages Web des éléments interactifs qui engagent un utilisateur.



Proprietes du JAVASCRIPT

JavaScript est la seule technologie de navigateur qui combine ces trois éléments :

  • Intégration complète avec HTML / CSS.
  • Les choses simples se font simplement.
  • Prise en charge par tous les principaux navigateurs et activée par défaut.

    • C’est ce qui rend JavaScript unique. C’est pourquoi il s’agit de l’outil le plus répandu pour créer des interfaces de navigateur.
    JavaScript permet également de créer des serveurs, des applications mobiles, etc...

    Ce que JavaScript PEUT faire dans le navigateur?

  • Ajouter du nouveau HTML à la page, changer le contenu existant, modifier les styles.
  • Réagir aux actions de l'utilisateur, exécuter sur les clics de souris, les mouvements du pointeur, les pressions de touches.
  • Envoyer des demandes sur le réseau à des serveurs distants et télécharger des fichiers
  • Obtenir et définir des cookies, poser des questions au visiteur, afficher des messages.
